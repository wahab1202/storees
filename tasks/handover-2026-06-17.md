## Deployments / how to ship (GWM box = goweldev)
- Backend: `storees-api` (pm2, port 4000, cwd `/var/www/html/storees-backend`) ‚Üí `gwm_storees`.
- Frontend: `storees-web` (pm2, port 4001, cwd `/var/www/html/storees-frontend`).
- **Use `scripts/deploy.sh`** (added this session) ‚Äî builds-first (aborts before restart on a bad
  build), frees the port (kills the zombie that caused `EADDRINUSE`), migrates, verifies `online`:
  ```bash
  cd /var/www/html/storees-backend  && bash scripts/deploy.sh backend  storees-api 4000
  cd /var/www/html/storees-frontend && bash scripts/deploy.sh frontend storees-web 4001
  ```
  Earlier the frontend went unstyled + crash-looped on `EADDRINUSE` from a stale `next start` +
  half-built `.next` ‚Äî the script prevents both. There's also a `Jenkinsfile` that runs it.
